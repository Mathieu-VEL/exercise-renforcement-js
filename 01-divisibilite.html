#01 - Divisibilité d'un nombre

Test de la divisibilité d'un nombre par 3.
Pour rappel, un nombre divisible par 3 est un nombre dont le reste de la division entière par 3 est nul. Ce nombre est un multiple de 3.


Par exemple,

"3, 6, 9, ..., 72, 75, 78, 81, ..."

sont des nombres divisibles par 3.

Etapes du programme

  1. Récupérer ce qui est saisi dans l'élément HTML <input type="text" ... />.
  2. S'assurer que ce qui est récupéré est bien d'un nombre.
  3. Faire le test de divisibilité.
  4. Afficher soit le résultat (alert (...)) sous la forme
    • Le nombre n est divisible par 3, si c'est divisible par trois
    • Le nombre n n'est pas divisible par 3, dans le cas contraire
    avec n qui contient la valeur testée saisie par l'utilisateur.
